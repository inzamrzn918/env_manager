# Change Log

All notable changes to the "env-manager" extension will be documented in this file.

## [0.0.1] - 2026-02-13

### Added
- Initial release
- Multi-project environment variable management
- Dual sync modes: Terminal injection and File sync
- Secure storage using VS Code SecretStorage
- TreeView sidebar with colored icons
- Full-featured Dashboard webview
- Import from existing `.env` files
- Backup protection for `.env` files
- Workspace-level variable overrides
- Modern UI with smooth transitions and shadows
